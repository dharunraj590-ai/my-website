?? Bootable USB – Windows

Download Windows ISO (Microsoft)
Insert 8GB or higher USB
Open Rufus
Select GPT (UEFI) or MBR (Legacy)
Click Start ? Bootable Ready

?? Bootable USB – macOS

Download macOS from App Store
Insert 16GB USB
Open Terminal
Use createinstallmedia command
Restart ? Hold Option ? Install
