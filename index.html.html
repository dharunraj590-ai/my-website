?? Bootable USB Guide

?? Windows OS

  • Download ISO from Microsoft.
  • Use Rufus tool to create bootable USB.
  • Select GPT for UEFI.
  • File system: NTFS or FAT32.
  • Click Start ? Wait until Done.

?? macOS

  • Download macOS DMG.
  • Insert USB (16GB+).
  • Open Terminal ? Use createinstallmedia command.
  • Wait until Terminal shows “Done”.
  • USB ready for macOS install.

?? Shortcuts

Windows Shortcuts

  • Win + E ? Open Explorer
  • Alt + Tab ? Switch Window
  • Win + L ? Lock PC
  • Ctrl + C ? Copy
  • Ctrl + V ? Paste

macOS Shortcuts

  • Cmd + Space ? Spotlight
  • Cmd + Tab ? Switch App
  • Cmd + C ? Copy
  • Cmd + V ? Paste
  • Cmd + Q ? Quit App
